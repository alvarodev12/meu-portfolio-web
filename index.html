Meus Projetos

Portfólio

Portfólio Web – Desenvolvido com HTML, CSS e JavaScript, este site apresenta meus projetos, habilidades e informações de contato de forma moderna e responsiva. Utiliza Flexbox e Grid para o layout, além de efeitos visuais e interatividade com JavaScript puro. O design foi inspirado em boas práticas de UI/UX, garantindo uma navegação fluida e intuitiva.

Web Dev Burguer

O Web Dev Burguer é um site que Desenvolvi para aprimorar minhas habilidades em desenvolvimento web, focando em HTML, CSS, JavaScript e boas práticas de UI/UX. Neste projeto, criei uma interface intuitiva e funcional para pedidos de hambúrgueres online. A interface inclui botões para mostrar todos os itens, mapear preços e até filtrar os resultados de acordo com a preferência do usuário. Além disso, cada item do menu está detalhado com preços e descrições visíveis.

DevMoveis

DevMoveis - Aplicação web desenvolvida em React que utiliza a API do The Movie Database (TMDb) para exibir informações sobre filmes. O projeto destaca conceitos como consumo de APIs REST, manipulação de estados no React e roteamento com React Router. Além disso, foram utilizadas bibliotecas como Axios para requisições HTTP e Styled Components para estilização da interface.
